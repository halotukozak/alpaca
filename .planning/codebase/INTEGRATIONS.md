# Integrations

## External Services
- **GitHub:** Hosting, CI (GitHub Actions), and documentation (GitHub Pages).
- **Maven Central / Sonatype Central:** Target for artifact publishing.

## Development Tools
- **Scala Steward:** Integrated via Mill plugin for automated dependency management.
- **Git:** Used for versioning (`VcsVersion`) and source links in Scaladoc.

## Documentation
- **Scaladoc:** Configured with custom logo and footer, linked to GitHub source.
- **GitHub Pages:** Expected hosting for documentation at `https://halotukozak.github.io/alpaca/docs/index.html`.
